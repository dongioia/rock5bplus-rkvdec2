# Kernel 7.1 Beryllium-Complete Rebuild — Implementation Plan

> **For agentic workers:** sequential, stateful kernel work — execute inline (superpowers:executing-plans), not parallel subagents. Steps use checkbox (`- [ ]`) syntax. Kernel work happens ONLY in the Docker volume `linux-beryllium-tree` via `docker exec dev-server` / `docker/run.sh`.

**Goal:** Build a clean, maximal-RK3588-hardware-support 7.1-final kernel for Rock 5B+, based on Collabora's pinned `rockchip-v7.1`, with our AV1-IOMMU fix folded in, deployed cleanly with per-version module dirs + a retained bootable fallback, then retest AV1 and (if positive) report to Collabora.

**Architecture:** Base = `collabora/rockchip-v7.1` (pinned, 167 commits on 7.1.0, full RK3588 enablement + vsi-iommu + Hantro AV1). Replay only our true downstream delta on top. Config = in-tree `beryllium_rk3588_defconfig` + a small fragment (`CONFIG_VSI_IOMMU=y`, `CONFIG_MODULE_ALLOW_BTF_MISMATCH=y`). Full `make clean` build (no incremental BTF desync). Deploy to dedicated `/boot/vmlinuz-linux-<tag>` + own initramfs + GRUB entry; keep the current `7.1.0-rc1-vsi+` kernel as fallback.

**Tech Stack:** Linux 7.1, Docker `rock5b-dev` (Arch ARM64), `docker/run.sh build-kernel`, GRUB, mkinitcpio, board Rock 5B+ (.157 eth / .160 wifi fallback).

## Global Constraints

- Kernel git/patch/make ONLY inside Docker volume `/k` (macOS case-insensitive corrupts netfilter). Host = `git push`/`remote` only.
- `INSTALL_MOD_STRIP="--strip-debug --keep-section=.BTF"` (never `=1`) — preserves BTF, else nfnetlink/r8169 fail.
- After ANY built-in (`=y`) config change → **full `make clean` rebuild** (incremental relinks vmlinux BTF but not modules → boot half-brick). `CONFIG_MODULE_ALLOW_BTF_MISMATCH=y` is the permanent safety net.
- Deploy modules to `/usr/lib/modules/<ver>/` (never `/lib/modules`; `/lib`→`usr/lib` symlink on Arch).
- Snapshot mandatory before build (`snapshot_guard.sh`); deploy overwrites single boot image → use per-tag image name.
- Keep exactly ONE known-good fallback kernel bootable at all times (current `7.1.0-rc1-vsi+`).
- AV1 fix centerpiece: `CONFIG_VSI_IOMMU=y` (binds `fdca0000.iommu` for Hantro AV1 → IOVA scatter-gather instead of CMA → no CMA exhaustion).
- Public Collabora report (Phase 6) = plain text, English, `humanizer` skill, no AI `Signed-off-by`, `Assisted-by:` trailer if patch attached.
